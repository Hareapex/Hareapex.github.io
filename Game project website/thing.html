My project game

why did decide to make this game?

It started with a video i watched

The main reason i wanted to start making this game is because of video made by beambuddy about the game called signalis, the game was pyschological horror story which had a great atmossphere and captivating storyline

The other reason why i wanted to start making this type of game is the video game called Darkwood which is a 2d top down survival horror game, where you try to survive each night by collecting supplies during daytime and hunkering down during the night to survive the horror of the night. What made me really like the game was the lack of jumpscares, i think its a super overused concept and is really boring, Darkwood did something diffrent... They made a great atmossphere that scares the player instead of the classical jumpscare.
